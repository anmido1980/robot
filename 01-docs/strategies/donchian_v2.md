# Стратегия: Donchian Breakout v2 (Turtle 20/10 + ADX>15)

**Дата:** 2026-06-15
**Статус:** paper (подготовка)
**MVP:** yes (кандидат на phase 4)

## Цель

Следовать тренду на Si/1h через пробой 20-периодного канала Дончиана.
ADX>15 отсекает сигналы в боковике. Закрытие по противоположному пробою
10-периодного канала или по ATR-стопу (2 ATR).

## Инструменты

- **Тикер:** Si (фьючерс на USD/RUB)
- **Class code:** SPBFUT
- **Минимальная ликвидность:** Si — самый ликвидный фьючерс ММВБ,
  среднедневной объём > 500k контрактов. Спред 1-2 тика.

## Таймфрейм

- Рабочий таймфрейм: **1 час**
- Время удержания позиции: среднее ~50 часов, максимум ~250 часов
  (по round 2: max 92 сделки за 5+ лет = ~18 сделок/год)
- Торговые сессии: 24/5 (фьючерсы Si торгуются почти круглосуточно,
  но ликвидность сосредоточена в 10:00–18:50 МСК)

## Вход

Условия входа (формализованные):

1. **Канал:** `upper = max(high[-20:-1])`, `lower = min(low[-20:-1])`
   (предыдущие 20 баров, исключая текущий)
2. **LONG:** `close > upper`
3. **SHORT:** `close < lower`
4. **Фильтр ADX:** `ADX(14) >= 15` (на тех же 1h свечах).
   Отсекает сигналы в чистом боковике.
5. **Объём:** ADX-фильтр обязателен; volume-фильтр опционален.

## Выход

- **Противоположный сигнал:** close < min(low[-10:-1]) для LONG
  (или close > max(high[-10:-1]) для SHORT) — `exit_period=10`
- **ATR-стоп:** `entry_price ± 2.0 * ATR(14)`. Активируется сразу при входе.
- Принудительного закрытия по времени нет — позиция удерживается до сигнала
  или стопа.

## Параметры (для paper-trading)

| Параметр | Значение | Описание |
|---|---|---|
| `entry_period` | 20 | Период канала входа |
| `exit_period` | 10 | Период канала выхода |
| `atr_period` | 14 | Период ATR для стопа |
| `stop_atr_mult` | 2.0 | Множитель ATR для стопа |
| `adx_min` | 15.0 | Минимальный ADX для входа |
| `adx_period` | 14 | Период расчёта ADX |
| `volume_min_ratio` | 0.0 | Volume-фильтр выключен |

Реализация: `02-source/strategies/swing/donchian_breakout_v2/strategy.py`.

## Управление позицией

- Макс. размер позиции: **1 контракт** (paper-старт)
- Пирамидинг: **нет** (новая позиция только после закрытия предыдущей)
- Хедж: **нет**

## Риск-параметры

- Макс. убыток на сделку: ~2 ATR (≈ 100-200 пунктов Si = 100-200 RUB)
- Макс. количество одновременно открытых позиций: **1**
- Ожидаемая частота сделок: ~18 сделок/год (по round 2)

Глобальные риск-лимиты (из RiskConfig, `02-source/risk/config.py`):

- `max_position_pct = 0.5` — позиция ≤ 50% депозита
- `max_daily_loss_pct = 0.05` — дневной убыток ≤ 5%
- kill-switch при −5% за день или ручной (файл `STOP.flag`)

## Границы применимости

- Работает на **Si/1h**, история 2021-2026.
- Edge связан с **долгосрочным трендом Si** (USD/RUB) — может
  исчезнуть при смене режима валютного рынка.
- ADX-фильтр > 15 — не отсекает в боковике, но и не слишком жёсткий.
- Volume-фильтр не используется (на 1h Si объёмы стабильны).

## Риски

1. **Переподгонка:** параметры (20/10/2 ATR) — стандартные для Turtle.
   ADX>15 — естественный порог. Нет тюнинга на OOS, edge выдержал 2 раунда.
2. **Режим рынка:** если Si войдёт в долгосрочный боковик (как 2023-W3),
   стратегия даст серию убытков. Kill-switch защищает на уровне −5%/день.
3. **Ликвидность:** при резком падении объёмов (holidays, cобытия)
   slippage вырастет. Backtest использует `slippage_ticks=1`.
4. **Корреляция:** стратегия лонг-биас/шорт-биас зависит от того,
   где находится Si в большом тренде. В 2025 — шорт-биазный год.
5. **Margin/ГО:** Si требует маржу ~3-5k RUB. Размер позиции в paper
   начинается с 1 контракта.
